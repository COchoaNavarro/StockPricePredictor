# StockPricePredictor
This program uses Long Short Term Memory (LSTM) to predict the closing stock price of Tesla (or any other Stock) using the last 60 day stock price between the dates 2015-01-01 to 2020-12-16.

## Requirements
 - Python 3.7 up
 - Jupyter Notebook
 - Numpy
 - Keras
 - MatplotLib
 - Pandas
 - Scikit & Scipy
